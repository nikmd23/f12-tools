Special Features

Google Chrome

  • Support for viewing WebSocket frames when a WebSocket request has been selected.
  • Sort the Waterfall Chart by Start Time, Response Time, End Time Duration or Latency.
  • Toggle for compressed or expanded Http Request information per row.

Mozilla Firefox

  • Content distribution chart aka "Performance Analysis".

Notable Omissions

Mozilla Firefox

  1. OnLoad & DomContentReady indicators.
  2. Request initiator information.
  3. Cookie details including: domain, path, expiration, security flags.
  4. Web sockets support.
